# BMI Calculator

This is my **first coding project** at age 17, built as part of my pre-university journey into Biomedical Engineering.  
It’s a simple Python program that calculates Body Mass Index (BMI), shows your category, and gives basic health advice.

## Features
- Input validation (checks for numbers and positive values)
- BMI calculation
- Categories: Underweight, Normal, Overweight, Obese
- Health advice for each category
- Beginner-friendly code structure

## How to Run
1. Install Python 3
2. Save the file as `bmi_calculator.py`
3. Run in terminal: `python bmi_calculator.py`
4. Enter your weight (kg) and height (m)
5. Get your BMI result + advice
